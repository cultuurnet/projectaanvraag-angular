Integratie via Widgets

De UiTwidgets van publiq zijn eenvoudige toepassingen waarmee je agenda-informatie in je website kan integreren.

De agenda-informatie komt uit de UiTdatabank en wordt gefilterd op maat van de scope van de website. Je kan de widgets volledig aanpassen aan de vormgeving van je website.

  • Geschikt voor basisoplossingen
  • Zelf of door publiq in te stellen
  • Een doorzoekbare agenda
  • Uitgebreide configuratiemogelijkheden: HTML, CSS en JS

Voorbeelden:
SeniorenNet, Toegankelijke UiTagenda van Inter of VivaVlaanderen

Integratie via Search API

De UiTdatabank API (JSON-LD) biedt developers toegang tot de rijkste en meest volledige vrijetijdsbron van Vlaanderen en Brussel. Bouw zelf applicaties zoals uitinvlaanderen.be, uitmetvlieg.be, cultuurkuur.be en de meer dan 500 andere partners die de UiTdatabank in hun platform integreerden.

  • Platformonafhankelijk
  • Individuele oplossingen op maat mogelijk
  • Bekroond met de award API van het Jaar

Voorbeeld: Open Monumenten

Evenementen aanmaken via entry API

De UiTdatabank entry API (JSON-LD) biedt de mogelijkheid om items in UiTdatabank aan te maken, aan te passen en te verwijderen.

  • JSON-standaard
  • Activatie op live-omgeving enkel op aanvraag
  • Bekroond met de award API van het Jaar 2019

Voorbeeld: het invoerformulier op uitdatabank.be
