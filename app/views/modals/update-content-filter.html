Contentfilter

Met een contentfilter bepaal je welke data er opgehaald wordt. Zo kan je voor je project het getoonde aanbod filteren op gemeente of type.

  • Maak gebruik van de Search API notatie.
  • Je kan je query testen via uitdatabank.be. Daar vind je ook een querybouwer.
  • Voorbeeld: filter alle films die getoond worden in Gent category_eventtype_name:Film AND category_flandersregion_name:Gent
  • Om enkel Belgisch of Nederlands aanbod te doorzoeken voeg je country:BE of country:NL toe aan je query.
  • Maak gebruik van 'advanced queries' binnen de Search API3 notatie. Je hoeft de q=niet te vermelden.
  • Voorbeeld: filter alle films die getoond worden in Gent terms.id:0.50.6.0.0 AND address.\*.postalCode:3000
  • Om enkel Belgisch of Nederlands aanbod te doorzoeken voeg je address.\*.addressCountry:BE of address.\*.addressCountry:NL toe aan je query.
