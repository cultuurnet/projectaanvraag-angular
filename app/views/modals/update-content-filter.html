Contentfilter

Met een contentfilter bepaal je welke data er opgehaald wordt. Zo kan je voor je project het getoonde aanbod filteren op gemeente of type.

  • Maak gebruik van de Search API notatie.
  • Je kan je query testen via uitdatabank.be. Daar vind je ook een querybouwer.
  • Voorbeeld: filter alle films die getoond worden in Gent "category_flandersregion_name:Gent AND category_eventtype_name:Film"
